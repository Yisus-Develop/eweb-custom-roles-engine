Professional Role & Capability management engine for WordPress. This plugin allows you to define, manage, and scale custom user roles with granular permission control. Part of the **EWEB Plugin Suite**.

## 🚀 Key Features

- **Granular Role Definition:** Create custom roles tailored to specific business needs.
- **Capability Management:** Precisely control what each user can see and do.
- **Security Hardening:** Limit access to sensitive admin areas based on custom roles.
- **Developer Friendly:** Extensible architecture to programmatically add or modify permissions.
- **Lightweight & Fast:** Minimal footprint on your WordPress installation.

## 🏗️ Technical Architecture

- **Clean Activation/Deactivation:** Handles role registration and cleanup without leaving ghost data.
- **Custom Admin UI:** Intuitive management of the "Operador" role and its derivatives.
- **GitHub Sync:** Integrated update system for seamless maintenance.

## 🛠️ Installation

1. Upload the `rol-operador-base` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Manage roles and permissions under the **Users > Roles** (or custom menu) section.

## 🔄 Updates

Supports automatic updates via `Yisus-Develop/eweb-custom-roles-engine`.
